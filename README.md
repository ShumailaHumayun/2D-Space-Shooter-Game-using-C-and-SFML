# 2D-Space-Shooter-Game-using-C-and-SFML
Developed a 2D Space Shooter Game using C++ and SFML as a learning project.The game features player-controlled jet movement, alien enemies, bullet shooting, collision detection, score tracking, sound effects, graphics, and increasing difficulty with more aliens spawning every 5 seconds. The jet has health points, and the game ends when all are lost
